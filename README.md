# Setup of mac


## HOMEBREW

```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### Packages

Darktable
````sh
brew install --cask darktable
```

Alacritty
```sh
brew install --cask alacritty
```
Tmux
```sh
brew install tmux
```
Nvim
```sh
brew install nvim
```
